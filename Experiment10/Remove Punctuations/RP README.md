
## Problem Statement
Write a Python program to remove all punctuation marks from a given string and print the resulting string.

---

## Algorithm
1.Start

2.Read string st

3.Find length n

4.Set flag = True

5.Compare characters from start and end

6.If mismatch → flag = False

7.If flag true → Palindrome else Not Palindrome

8.Stop
