## Problem Statement
Write a Python program to perform the multiplication of two square matrices. The program should take two matrices as input and output their product

---

1️⃣ Input the size n of the square matrix.
2️⃣ Input elements of Matrix A (n × n).
3️⃣ Input elements of Matrix B (n × n).
4️⃣ Create a result matrix R with all elements 0.
5️⃣ Multiply matrices using nested loops:

R[i][j] = R[i][j] + A[i][k] * B[k][j]
6️⃣ Print the result matrix R.
